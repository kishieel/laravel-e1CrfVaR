Introduction

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

To authenticate requests, include an Authorization header with the value "Bearer {YOUR_ACCESS_TOKEN}".

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

Auth

POST api/auth/sign-up

Example request:
curl --request POST \
    "http://localhost/api/auth/sign-up" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"email\": \"augusta.nolan@example.com\",
    \"title\": \"zglqwgfrsu\",
    \"password\": \"d9R|%@D-!jD5O\\/*q-\",
    \"first_name\": \"wtsoorbytbsupb\",
    \"last_name\": \"cvymlny\"
}"

const url = new URL(
    "http://localhost/api/auth/sign-up"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "email": "augusta.nolan@example.com",
    "title": "zglqwgfrsu",
    "password": "d9R|%@D-!jD5O\/*q-",
    "first_name": "wtsoorbytbsupb",
    "last_name": "cvymlny"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

GET api/tasks

requires authentication

Example request:
curl --request GET \
    --get "http://localhost/api/tasks?page=14&limit=11&search=beatae&sort%5B%5D=omnis" \
    --header "Authorization: Bearer {YOUR_ACCESS_TOKEN}" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost/api/tasks"
);

const params = {
    "page": "14",
    "limit": "11",
    "search": "beatae",
    "sort[]": "omnis",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Authorization": "Bearer {YOUR_ACCESS_TOKEN}",
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());

Query Parameters

page   integer  optional

search   string  optional  

Search query to filter tasks by title, description or status. Example: beatae

sort[]   string  optional  

Sort users by columns and direction (e.g., sort[]=name:asc). Example: omnis
